## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+ installed
- Git installed
- WhatsApp account

### Installation

```bash
# Clone the repository
git clone https://github.com/Eliahhango/EliTechWiz-GENZ.git
cd EliTechWiz-GENZ

# Install dependencies
npm install --legacy-peer-deps

# Configure settings
# Edit settings.js with your details

# Run the bot
node index.js
```

### First Time Setup

1. **Scan QR Code** - When bot starts, scan the QR code with WhatsApp
2. **Or Use Pairing Code** - Use `.scan` command for pairing code
3. **Join Channel** - Required to use bot commands
4. **Verify** - Use `.verifyjoin` after joining channel
